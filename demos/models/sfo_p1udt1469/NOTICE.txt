This model is based on data collected from [1].
It represents a single feeder from a much larger collection with the following changes:

* Removed disconnected lines to eliminate zero voltage buses
* Used openDSS save circuit to export cleaner model files
* Renamed master.dss to main.dss
* Created pu demand profile based on full year run using all profiles from total S
* Created average solar profile from all provided solar profiles
